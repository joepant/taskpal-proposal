Task Pal

Team members: Joe Pantaliono

Task Pal will be a convenient task tracking app storing things to do, notes on the task, and status of completion.

I'll be using HTML/CSS/JS, Node.js, Azure, and MongoDB

This proposal was last updated 6/8/2022

Feature list:

  • Front-end focused on user experience. Allow for adding, updating, deleting/completing, and editing of tasks
  • Separate page for creating a new task
  • Use MongoDB to store tasks
  • Host on Azure with GitHub and Node

Future features:

  • Color coordination of notes
  • User accounts/auth

Basic technical features:

  • Front end: use HTML/CSS and JavaScript to create a smooth front end
  • Cloud hosting and CI/CD: use the same workflow as most of our projects in class, leveraging Azure and GitHub
  • MongoDB: although we haven’t covered this quite yet, I looked ahead to see if we’d be doing any database work and chose based on the syllabus

Advanced technical features:

  • Personal accounts/authentication
